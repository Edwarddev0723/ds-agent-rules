# Recipes Overview

Recipes are ready-to-use `.ai-rules.yaml` configurations designed for specific real-world use cases. Instead of guessing which snippets to combine, you can copy a recipe that matches your project goals.

## Available Recipes

| Recipe | Best For | Profile | Key Snippets |
|--------|----------|---------|--------------|
| [Enterprise RAG](rag-chatbot.md) | Chatbots, Knowledge Retrieval | `llm-eng` | `rag`, `prompt-engineering`, `responsible-ai` |
| [AI Agent System](ai-agent-system.md) | Multi-agent, Tool-using LLMs | `llm-eng` | `agentic-ai`, `prompt-engineering`, `mlops` |
| [Data Pipeline](data-pipeline.md) | ETL, Feature Stores | `data-eng` | `streaming-ml`, `mlops`, `tabular-ml` |
